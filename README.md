# PROGRAMACIÓN DE VIDEOJUEGOS EN TERMINAL CON C++

Estas son las notas, ejemplos y proyecto del curso de C++ de Platzi.
El curso está algo obsoleto en cuanto a que tiene más de 6 años, pero en las notas e ido poniendo actualizaciones y comentarios sobre las últimas versiones de c++

El curso es una introducción a la programación de videojuegos en terminal con C++. Se cubren temas como:

- Fundamentos de C++
- Estructuras de datos
- Programación orientada a objetos
- Diseño de videojuegos
- Implementación de un juego de aventuras en terminal

El proyecto final es un juego de aventuras en terminal donde el jugador puede explorar un mundo, interactuar con objetos y personajes, y completar misiones.

## Requisitos

- Conocimientos básicos de programación
- Un entorno de desarrollo para C++ (como Visual Studio, Code::Blocks, o un editor de texto con un compilador de C++)
- Ganas de aprender y divertirse programando

## Contenido del curso

1. Introducción a C++
2. Variables y tipos de datos
3. Estructuras de control
4. Funciones
5. Arreglos y vectores
6. Punteros y referencias
7. Programación orientada a objetos
8. Diseño de videojuegos
9. Implementación del juego de aventuras
10. Mejoras y optimizaciones

## Conclusión

Este curso es una excelente oportunidad para aprender a programar videojuegos en terminal con C++. A través de ejemplos prácticos y un proyecto final, los estudiantes podrán aplicar sus conocimientos y desarrollar sus habilidades en programación. ¡No pierdas la oportunidad de crear tus propios videojuegos y divertirte programando!
